#  ToDo_DAW3 — Gerenciador de Tarefas  
Aplicação completa com **Backend em Spring Boot** e **Frontend em Blazor WebAssembly**.  
Permite adicionar, listar, marcar como concluída e excluir tarefas.

---

##  Tecnologias Utilizadas

### **Backend (API REST)**
- Java 17+
- Spring Boot 3.x
- Spring Web
- Spring Data JPA
- H2 Database (ou outro caso tenha alterado)
- Maven

### **Frontend**
- Blazor WebAssembly (.NET 8)
- C#
- Bootstrap 5

---

#  Como Rodar o Projeto

---

##  **1. Rodando o Backend (Spring Boot)**

Entre na pasta **api**:

```
cd api
mvn spring-boot:run

```
##  **2. Rodando o Frontend (Blazor)**

Entre na pasta do projeto Blazor:
```
cd frontend-blazor/TaskManager
dotnet run

```
A aplicação abrirá em:
http://localhost:5041
